## Integration Points

1. **Sidebar Navigation**: New button between Search and Settings
2. **Jira API**: Uses existing Jira service for issue data
3. **Settings**: Respects existing Jira connection settings
4. **External Links**: Opens Jira issues in system browser
5. **Database**: Separate SQLite database for kanban data
